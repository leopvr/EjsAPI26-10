Ejercicios JS 26-10

Utilizando la respuesta de users de JSONPlaceholder - Recurso #1 -, mostrar el resultado por consola utilizando template strings de los siguientes pedidos:

  1. Recorrer el array, recuperar el nombre del usuario y asignarle su id.
  2. Recorrer el array y armar un texto mostrando el nombre del usuario, su correo y su teléfono.
  3. Recorrer el array y armar un texto mostrando el nombre de su compañía y su frase clave.
  4. Mientras el id de usuario sea menor que 5, mostrar la localidad con su latitud y longitud
  5. Hacer un switch que le pida al usuario ingresar números comprendidos entre el 1 y el 10 (correspondiente a los id) y, acorde al dato ingresado, mostrar toda la información del user seleccionado en una oración con coherencia y cohesión
  6. Al finalizar las primeras cinco actividades:
    1. Remover el último elemento
    2. Agregar un elemento con las mismas keys al final
    3. Remover el primer elemento
    4. Agregar un elemento con las mismas keys al principio
    5. Buscar un elemento que tenga el username “kamren” y, si existe, devolver el nombre, el username y el correo. Para la búsqueda del username, implementar los métodos toLowerCase() o toUpperCase()

Utilizando la respuesta de products de FakeStoreApi- Recurso #2 -, mostrar el resultado por consola utilizando template strings de los siguientes pedidos:

  1. Recorrer el array, recuperar el nombre del producto y asignarle su id.
  2. Recuperar el nombre del producto, la categoría, el precio en usd y el rate que tiene dicho producto.
  3. Mostrar el nombre y precio de los productos que tienen un rate entre 1.8 y 3.2
  4. Mostrar el nombre y precio de los productos que tienen un precio mayor a 150
  5. Mostrar el nombre y rate de los productos que tienen un precio menor a 150

Utilizando la respuesta de characters de Rick & Morty API - Recurso #3 -, mostrar el resultado por consola utilizando template strings de los siguientes pedidos:

  1. Mostrar el nombre del personaje, su especie, el nombre de su lugar de origen y la cantidad episodios en los que aparece
  2. ¿Cuáles son las claves que figuran dentro de la clave “info”? Buscar el método aplicable a objetos que permite mostrar esta info
  3. ¿Cuáles son los valores de las claves mostradas en el ítem anterior? Buscar el método aplicable a objetos que permite mostrar esta info
